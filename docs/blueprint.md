# **App Name**: Rally Klasikoak

## Core Features:

- Erabiltzaile autentifikazioa: Erabiltzaileak erregistratu, saioa hasi eta pasahitza berreskuratu ahal izango du.
- Produktuen katalogoa: Produktuak kategoriaren arabera antolatuta egongo dira, prezioekin, deskribapenekin eta irudiekin.
- Saskia eta Eskaerak: Erabiltzaileek saskian produktuak gehitu eta eskaerak sortu ahal izango dituzte. Pasarela de pago para introducir la tarjeta de credito

## Style Guidelines:

- Kolore nagusia: Gorri bizia (#E53935), kiroltasuna eta pasioa adierazteko.
- Atzeko planoko kolorea: Gris argia (#F5F5F5), diseinu garbia eta modernoa izateko.
- Akzentu kolorea: Urdin elektrikoa (#1E88E5), botoiak eta nabarmentzeko eremuak markatzeko.
- Letra-tipoa: 'Space Grotesk', sans-serif motakoa, izenburuetarako eta gorputz-testurako. Oharra: Google Fonts soilik onartzen dira.
- Ikono sinpleak eta argiak erabiliko dira, produktuaren kategoria adierazteko.
- Diseinu argia eta erraz nabigatzeko modukoa erabiliko da, esperientzia intuitiboa eskaintzeko.
- Animazio sotilak erabiliko dira interakzioetan, erabiltzailearen esperientzia hobetzeko.
